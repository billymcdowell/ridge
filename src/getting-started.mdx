## Quick Start

Get Ridge UI running in under 2 minutes:

### 1. Install
```bash
npm install ridge-ui
```

### 2. Import
```javascript
import 'ridge-ui';
```

### 3. Use
```html
<rdg-button variant="primary">Hello Ridge UI!</rdg-button>
```
